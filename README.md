## Encontrando o Maior, Menor e Média de 10 Números

### Descrição
Este script Python lê 10 números inteiros positivos do usuário e calcula as seguintes estatísticas:

* **Maior valor:** O maior número digitado.
* **Menor valor:** O menor número digitado.
* **Média:** A média aritmética dos números.

### Como usar
1. **Salve** o código como um arquivo Python (por exemplo, `estatisticas.py`).
2. **Execute** o arquivo no seu terminal usando o comando `python estatisticas.py`.
3. **Digite** os 10 números quando solicitados.
4. **O resultado** será exibido na tela.
